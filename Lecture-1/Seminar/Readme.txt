IP-адрес    -   числовой идтфикатр (именой (на одно железо один номер))

IPv4-15.252.12.0    -   только цифры 255 самое болшое
IPv6-2001:0db8:85a3:0000:0000:8a2e:0370:7334    -   цифры и буквы

Иза не хватки IPv4 зделали IPv6

Раздоет провайдер они миняються (диномические). Можно заказать у провайдера статический IP-адрес (за оплату)

15.252.12.0:5000    - это IP-адрес и порт определеноо приложения

                80 и 443 порты используются для сайтов

15.252.12.0:80      -   это http служба, которая отвечает вам при запрашивании страниц сайта   
15.252.12.0:443     -   https зашифрованный вариант http, с использованием SSL сертификатов.

DNS (англ. Domain Name System «система доменных имён») — компьютерная распределённая система для получения
информации о доменах. Чаще всего используется для получения IP-адреса по имени хоста (компьютера или
устройства), получения информации о маршрутизации почты и/или обслуживающих узлах для протоколов в домене
(SRV-запись).
